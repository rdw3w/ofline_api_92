# 🚀 Offline AI API System

## ✅ Features
- Model store
- Smart model selection
- Model download system
- Fully offline chat support

## 🛠️ Setup

```bash
cd backend
npm install
npm start
```

## 🔗 API Endpoints

### Get models
GET /models

### Best model
GET /best-model?storage=5000

### Download model
GET /download?name=tinyllama
